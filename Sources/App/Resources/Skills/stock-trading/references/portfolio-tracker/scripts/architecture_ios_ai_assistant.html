Hermes Personal AI Assistant

System Architecture — iOS Client + Backend + Knowledge Graph

System Overview

Client (iOS App)

  • SwiftUI + native iOS integrations (Photos, Maps, Siri)
  • Obsidian-like vault with Space/Folder hierarchy
  • Markdown editor with WYSIWYG preview
  • Screenshot → OCR → MD conversion plugin
  • Command palette: kb-compile, kb-query, kb-remind
  • Offline-first with CloudKit sync

Backend (Hermes-Powered)

  • Vapor/Hummingbird API (Swift or Go)
  • Multi-tenant: each user = isolated vault
  • Hermes agent handles kb-compile, kb-query
  • LLM routing (Claude for analysis, local for simple queries)
  • Plugin execution sandbox (Docker)
  • Scheduled compilation & digest generation

Knowledge Layer

  • PostgreSQL: structured data (spaces, files, metadata)
  • pgvector: semantic search over embeddings
  • Graph edges: causal relationships, timelines
  • S3/MinIO: original media blobs
  • Redis: cache compiled context, sessions
  • Automated indexing pipeline
